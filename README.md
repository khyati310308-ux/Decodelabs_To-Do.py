📝 To-Do List Application

📖 Description

The To-Do List Application is a simple command-line project developed using Python that helps users organize and manage their daily tasks efficiently. Users can add new tasks, view all saved tasks, and keep track of their work through an interactive menu-driven interface. This project is designed to strengthen fundamental Python programming concepts while solving a real-world problem.

---

🎯 Project Objectives

- Develop a basic task management application using Python.
- Practice working with lists, loops, and conditional statements.
- Understand menu-driven programming.
- Improve problem-solving and logical thinking skills.

---

✨ Features

- ➕ Add new tasks
- 📋 View all added tasks
- 🖥️ Interactive menu-driven interface
- ⚡ Fast and lightweight application
- 👨‍💻 Beginner-friendly project
- 🔄 Runs continuously until the user chooses to exit

---

🛠️ Technologies Used

- Programming Language: Python 3
- IDE: Pydroid 3
- Platform: Android
- Libraries Used: Python Standard Library (No external libraries)

⚙️ How It Works

1. The application displays a menu with different options.
2. Users can choose to add a new task.
3. All entered tasks are stored in a list.
4. Users can view all saved tasks at any time.
5. The application continues running until the user selects the Exit option.

---

▶️ How to Run

1. Install Pydroid 3 on your Android device.
2. Open the project folder in Pydroid 3.
3. Run the "todo.py" file.
4. Follow the on-screen instructions to manage your tasks.
